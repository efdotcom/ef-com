This is a react application, it uses vite to run, i.e. don't change the vite configs otherwise it might not build or run correctly.

to run:

1. ctrl + shift + ` 
2. npm install 
3. npm run build

then to preview current site

4. npm run preview

OR to spin up a local development environment 

5. npm run dev

Do the latter if you are actively making changes yourself, do the former if you just want to check how it looks after I've made some commits
